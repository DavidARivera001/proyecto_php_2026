# Proyecto_php

## Descripción:

mira el proyecto de php sobre la carga de una imagen.

Cuenta con los siguientes prerequisitos:
-Contar con una maquina virtual con Linux Ubuntu.
-Intalarle a esa maquina: php, apache2, C, git y github.

Desde la maquina local (fisica) instala en Visual Studio Code, la extención de SSH Remote.
Sigue los pasos de configuración habitual
y agrega la IP de la computadora Virtual que utlizaras.

Desde la maquina local, ingresa a la Extencion SSH Remote, selecciona la IP y coloca la contraseña del equipo virtual.
Podras enviar comandos desde la terminal para instalar los prerequisitos mencionados.

Asimismo, busca la carpeta de php para crear el primer archivo, alli podras programar el codigo que necesites.
Agregar la imagen y texto a discreción del usuario.

Por ultimo completar los pasos de GIT
Crear el repositorio web y realizar la carga al respoitorio.
